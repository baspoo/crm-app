appApi : sdk สำหรับติดต่อ server ของฉัน เป็น script กลาง ห้ามเอาของเฉพาะโปรเจ็ค crm เข้าไปใส่ ฉันจะเก็บไว้ใช้ยาวๆกับงานอื่นๆ เช่น callApi , callApiInternal จะใช้ crmApi เข้ามาเรียกใช้เป็นหลัก 
crmApi : เป็น api ตัวเสริมปรับแต่งสำหรับ crm โดยเฉพาะ และ function ทุกอย่างของ crm เช่น login , redeem , getreward มีเฉพาะ apiและเก็บ current data เท่านั้น 
จัดการ state , function , model schema ทุกอย่างของแอฟ crm นี้
pageManager : จัดการระบบ page ทั้งหมด



ทำการ await app.init(); เพื่อเช็คทุกอย่าง login? getAppData? getTheme? แต่ตอนนี้แค่ awiat 1sec แล้ว return เลย เราจะมาทำกันภายหลัง
เพื่อ init เสร็จ เปิด mainConsole ขึ้นมาเป็น หน้าหลัก เริ่มต้น

5. demoIframePage.html


ฉันจะยังไม่ทำ app.js แยกออกมา แต่ให้แต่ละหน้ารับผิดชอบ js ของตัวเองไปก่อนเพราะยากต่อการจัดการในตอนนี้
แต่ให้ทุกหน้า ทำ template สำหรับ ui ที่ต้องแก้ไข เพื่อให้ js ดึงค่าไปแก้ อย่า ให้ js พ่น div เข้าไปเอง มันจะเปลี่ยนสไตล์ยาก 
ให้ js ของแต่ละหน้า จัดการ function ของแต่ละหน้าไปเอง แต่เรียก api จาก crmApi ไปใช้

สร้าง ทุกหน้า ขึ้นมาใหม่ ยกเว้น appApi.js ใช้ของเดิมของฉัน และเติม function test สำหรับให้แต่ละหน้า สามารถลองเรียกฟังชั่นใน crmApi ว่ามาถึงไหม และสามารถดึงค่าจาก crmApi พวก currentData ไปแสดงผลได้ไหม
สร้าง element สำหรับ test เยอะๆ พวก dropdown / grid item / btn / textbox / สกอบาร์ / panel
เพื่อทดสอบ template ที่ซ่อนไว้ และ ถูกเปลี่ยนจาก js ภายในเพจ 

เติมคำอธิบายให้ฉัน จะได้เข้าใจว่ามันทำงานยังไง สำหรับเรียนรู้ และวางแผนสำหรับทำต่อไป

ส่งออกมาทุกคริป เพื่อให้ฉัน ก๊อปปี้ไปใช้ได้เลย

ต้องการทำ Global Override
เอาไว้ที่ style.css เป็น ไฟล์ CSS หลักของแอป 
แล้วเขียน CSS ดั้งเดิมเข้าไปทำการ "หักดิบ" ครอบทับคลาสเดิม


เช่น
/* บังคับปุ่มทุกตัวในแอปให้กลายเป็นสไตล์ใหม่ทันที */
button, .btn, [role="button"] {
    border: 3px solid #000000 !important; /* บังคับใส่เส้นขอบสีดำหนา */
    box-shadow: 4px 4px 0px #000000 !important; /* บังคับใส่เงาสีดำแบบเหลี่ยม */
    border-radius: 8px !important; /* ปรับความโค้งตามต้องการ */
    transition: all 0.2s ease;
}

/* เอฟเฟกต์ตอนเอาเมาส์ไปวาง (Hover) */
button:hover, .btn:hover {
    transform: translate(-2px, -2px);
    box-shadow: 6px 6px 0px #000000 !important;
}

/* บังคับการ์ด หรือกล่องเนื้อหาต่างๆ ในแอป */
.card, .box, div[class*="shadow"] {
    border: 3px solid #000000 !important;
    box-shadow: 6px 6px 0px #000000 !important;
    border-radius: 12px !important;
    background-color: #ffffff !important;
}
